## Student Manager (C++ CLI)

### Mô tả
Ứng dụng quản lý danh sách sinh viên đơn giản chạy trên terminal.

### Cấu trúc dữ liệu sử dụng

1. Linked List
- Lưu danh sách sinh viên
- Dễ thêm/xóa

2. Stack
- Lưu lịch sử xóa
- Hỗ trợ Undo

3. Queue
- Lưu danh sách chờ
- Hoạt động theo FIFO

### Compile và chạy

```bash
g++ -std=c++17 main.cpp -o app
./app

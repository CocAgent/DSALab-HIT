# 📘 Chương 2: Tìm kiếm & Sắp xếp (C++)

## 📌 Giới thiệu

Project này cài đặt các **giải thuật cơ bản** trong môn *Cấu trúc dữ liệu & Giải thuật*:

* 🔍 Tìm kiếm
* 🔄 Sắp xếp

Phù hợp cho sinh viên học và ôn tập.

---

## ⚙️ Các thuật toán đã cài đặt

### 🔍 Tìm kiếm

* Linear Search (Tìm kiếm tuyến tính) — O(n)
* Binary Search (Tìm kiếm nhị phân) — O(log n)

### 🔄 Sắp xếp

* Selection Sort — O(n²)
* Insertion Sort — O(n²)
* Bubble Sort — O(n²)
* Quick Sort — O(n log n)
* Heap Sort — O(n log n)

## 🚀 Cách chạy chương trình

### 1. Clone repo

```bash
git clone https://github.com/your-username/your-repo.git
cd your-repo
```

### 2. Biên dịch

```bash
g++ Chuong2_TimKiem_SapXep.cpp -o main
```

### 3. Chạy chương trình

```bash
./main
```

*(Windows dùng `main.exe`)*

# 🚀 Danh Sách Liên Kết Đơn (Chương 3)

## 📌 Nội dung

Cài đặt **Danh sách liên kết đơn (Singly Linked List)** với các chức năng:

### 🔹 Thêm phần tử

* 1. Thêm vào đầu (ThemDau)
* 2. Thêm vào cuối (ThemCuoi)
* 3. Thêm sau node q (ThemSau)
* 4. Thêm trước node q (ThemTruoc)

### 🔹 Xóa phần tử

* 5. Xóa đầu (XoaDau)
* 6. Xóa cuối (XoaCuoi)
* 7. Xóa phần tử có giá trị x (XoaGiaTri)

### 🔹 Duyệt & xử lý

* 8. Duyệt danh sách (InDanhSach)
* 9. Tìm phần tử lớn nhất (TimMax)
* 10. Tìm phần tử x (TimX)
* 11. Tìm số chẵn đầu tiên (TimChanDau)
* 12. Tìm số chẵn cuối cùng (TimChanCuoi)
* 13. Liệt kê số âm (LietKeAm)
* 14. Tính tổng số dương (TongDuong)
* 15. Đếm số âm (DemAm)
* 16. Kiểm tra số chính phương (CoSoChinhPhuong)
* 17. Đếm phần tử cực đại (DemCucDai)

g++ main.cpp -o main
